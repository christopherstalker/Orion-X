# apps/site

This folder hosts the website frontend inside the Orion-X monorepo.

## If you already have an existing site

Copy your existing website files into this folder and merge/update `package.json` scripts to keep:

- `dev`
- `build`

## Local commands

```bash
pnpm --filter @orion-x/site dev
pnpm --filter @orion-x/site build
```
